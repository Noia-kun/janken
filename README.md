# Janken (Rock Paper Scissors)

A simple browser-based Rock Paper Scissors game built with vanilla JavaScript, HTML, and DOM manipulation.

🎮 **Play it here:** https://noia-kun.github.io/janken/

## How to Play

1. Open `index.html` in your browser.
2. Click **Rock**, **Paper**, or **Scissors**.
3. The computer picks randomly and the result of each round is displayed.
4. First to 5 points wins the match. Buttons disable automatically when the game ends.

## Features

- Click-based UI (no prompts/alerts)
- Live round result and running scoreboard
- Automatic win detection at 5 points

## Files

- `index.html` — page structure and buttons
- `script.js` — game logic and DOM updates
- `styles.css` — styling (if applicable)

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (no frameworks/libraries)
